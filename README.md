## Features

- **NTFS Hard Link Deduplication** — Unchanged files are hard-linked across snapshots, saving disk space
- **Live Streaming Backup Engine** — Real-time file backup with smooth progress tracking
- **Dark Mode** — Beautiful light/dark theme toggle with glassmorphism design
- **Automated Scheduling** — Configurable backup intervals (1-168 hours) with catch-up logic
- **Self-Binding Licensing** — Privacy-first RSA-PSS licensing with 14-day trial
- **One-Click Updates** — Automatic update checking every 12 hours
- **System Tray Integration** — Context-aware tray menu with pause/resume/skip controls
- **Retention Policies** — Conservative, Balanced, or Keep More History presets

## System Requirements

- Windows 10/11 (64-bit)
- NTFS file system
- .NET 9 Runtime (bundled in installer)
